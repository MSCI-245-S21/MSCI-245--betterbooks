#betterbooks

This is the web page that can user create their favorite books. Multiple users can upload multiple books. 

There are three sections on page, **Authors**, **Books**, **Users**. 

Authors, Books, Users can be created, edited, and delted. 

To create a **new book**, information about title, author's name, Year of publication, genre (fiction/non-fiction) are needed. 

To add a **new user**, user name and email are needed. Email cannot be duplicated with others. 



Authors - can create, edit, delete

books- year, author, genre - edit, delete, add

users- can create, no duplicate email 

Genre대로 나열할수 있음 fiction먼저, non-fiction나중에

Author이름 알파벳 순으로 나열가능
Book 발행연도 작은 숫자 - 높은 숫자 순으로 나열 가능
